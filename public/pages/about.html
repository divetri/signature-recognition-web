Tentang Web Presensi

Web ini digunakan untuk menampilkan hasil penelitian tentang sistem presensi. Teknologi yang digunakan adalah berupa identifikasi citra tanda tangan menggunakan Speeded-Up Robust Feature dengan pencocokan menggunakan FLANN
Proses presensi menggunakan Raspberry Pi 3 dan dokumentasi dengan realtime database menggunakan Pyrebase
